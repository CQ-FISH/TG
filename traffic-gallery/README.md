# 交通图库网站使用说明

## 项目简介

交通图库是一个全品类交通摄影作品分享平台，涵盖铁路、航空、陆运、水运、特殊交通等多个领域的摄影作品。本项目采用纯HTML5 + CSS3 + 原生JavaScript开发，无框架依赖，可直接部署使用。

## 核心功能

- **作品展示**：首页精选、最新/热门作品瀑布流、分类专题展示、作品详情大图预览
- **分类体系**：6大主分类+多级子分类，支持主/子分类联动筛选、标签精准检索
- **搜索功能**：全字段关键词搜索，支持标题、描述、标签、拍摄地点、摄影师多维度匹配
- **作品上传**：前端图片预览、分类联动选择、作品信息填写、本地数据持久化存储
- **交互功能**：作品点赞、浏览量统计、相关作品推荐、悬停动效、响应式适配
- **用户系统**：登录/注册功能，管理员权限控制
- **日志功能**：记录用户操作日志（登录、上传作品、评论、点赞等）
- **基础页面**：首页、图库筛选页、作品详情页、作品上传页、登录/注册页、404错误页

## 技术栈

- **核心技术**：纯 HTML5 + CSS3 + 原生JavaScript（无框架依赖）
- **适配目标**：兼容Chrome、Firefox、Edge、Safari主流浏览器，兼容IE11降级展示
- **数据存储**：前端使用 `localStorage` 实现数据持久化，可无缝对接后续后端接口
- **性能要求**：图片懒加载、代码轻量化、响应式布局，适配PC/平板/手机全设备

## 如何运行

### 方法一：直接打开

1. 下载本项目到本地
2. 直接双击 `index.html` 文件，在浏览器中打开即可

### 方法二：本地服务器（推荐）

1. 下载本项目到本地
2. 在项目根目录启动本地服务器
   - 使用 Python 3: `python -m http.server 8000`
   - 使用 Node.js: `npx http-server -p 8000`
   - 使用 PHP: `php -S localhost:8000`
3. 在浏览器中访问 `http://localhost:8000`

## 使用说明

1. **浏览作品**：在首页或图库页浏览作品，点击作品卡片进入详情页
2. **筛选作品**：在图库页使用分类、标签、关键词进行筛选
3. **上传作品**：点击导航栏的"上传"按钮，填写作品信息并上传图片
4. **登录/注册**：点击导航栏的"登录/注册"按钮，进行用户登录或注册
5. **点赞作品**：在作品详情页点击点赞按钮，为作品点赞
6. **搜索作品**：在导航栏搜索框输入关键词，点击搜索按钮

## 管理员功能

使用用户名 `admin` 登录，即可获得管理员权限，可查看所有用户操作日志。

## 注意事项

1. 本项目使用前端 `localStorage` 存储数据，刷新页面后数据会保留
2. 图片上传功能使用前端本地存储，实际项目中应对接后端存储服务
3. 登录/注册功能为模拟实现，实际项目中应对接后端认证服务
4. 由于使用前端存储，数据量较大时可能会影响性能
